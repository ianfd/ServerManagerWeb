Install a mysql server


If you want to store data in an SQL database, mostly you are going to use either the default MySQL-Server or a MariaDB-Server. In this tutorial we are going to install a MariaDB-Server.

!! A MariaDB-Server or MySQL-Server is required to use the ServerManager plugin. !!

There are a lot of ways to install an MySQL-Server. Down there you will find tutorials for your OS. Out there are a lot of good tutorials for installation, so I'm not going to write one on my own.
