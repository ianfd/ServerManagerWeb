TUTORIALS

Install the plugin


1. Step - Download the file:

Download the latest version of the ServerManager. You can to this either from the SPIGOT WEBSITE or directly from me: TBA

2. Step - Move into the plugins folder:

Drag the downloaded file into the plugins folder of your BungeeCord server. Make sure that you delete all older versions of the plugin that are in the plugins folder.

3. Step - Configure your plugin:

Setup MySQL / MariaDB:

If you don't already have a MySQL or MariaDB database, you will have to create one. Install MySQL

Before you can edit the plugins/ServerManager/config.yml you have to start the BungeeCord server one time, so that the config file is generated.

It's also good if you have the credentials of the mysql user, you want to use, ready.

After this file has been generated you have to open it. If you are on debian/ubuntu use nano or vim to open the config file of ServerManager. If you are on windows/mac, you can use the default text-editor.

After that you will see some variables you can edit. The fields you have to edit to get the ServerManager working, are the settings for MySQL.
You can find the MySQL settings at the line: 11, 13, 15, 17 and 19.

The rest of the settings can be changed by you!

After saving the file and a restart, the plugin is ready to be used.

Have a look at create your first server to find out more!
